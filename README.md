# Simple Express Typescript application

## Setup

### Clone the repo

### Change into preoject directory

```
cd node-express
```

### Install dependencies

```
pnpm i
```

### Build the project

```
pnpm build
```

or for dev environment using nodemon

```
pnpm watch
```

### Start the server

```
pnpm start
```

or for dev server

```
pnpm dev
```
